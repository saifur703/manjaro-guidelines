# First Steps After Fresh Manjaro Install

### Switch to Fastest Mirror
```
sudo pacman-mirrors --fasttrack
```

### Update Manjaro
```
sudo pacman -Syyu
```

### Install AUR package manager
```
sudo pacman -S yaourt
```
